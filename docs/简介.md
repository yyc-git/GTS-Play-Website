# 简介

【巨大娘的玩耍】是一款肉鸽H5 3D巨大娘动作射击游戏，您可以通过模组在各个方面扩展游戏


[进入游戏](https://gts-play.cn) • [进入游戏（调试模式）](https://gts-play.cn?mod)



## 开发环境准备

- 通过[模组开发Github](https://github.com/yyc-git/GiantessPlay-Mod?tab=readme-ov-file#%E5%B7%A8%E5%A4%A7%E5%A8%98%E7%9A%84%E7%8E%A9%E8%80%8D%E6%A8%A1%E7%BB%84)或者[直接下载模组开发项目](https://pan.baidu.com/s/1gn6hB0RV-k6FnG1to5R7VQ?pwd=1111)获取项目代码
- 安装 node.js(版本22及以上)
- 在项目根目录中执行`npm install`



## 项目结构解析

```plaintext
├── mods/   # 模组文件夹
├── protocols/   # 模组对应的协议（指定了其中的服务、数据的类型）文件夹
```

所有模组都存放在 `mods/` 文件夹中，每个模组都是一个独立的项目，使用TypeScript编写，包含以下主要文件：

- `package.json` - 模组的配置文件，包含模组名称、版本、作者、依赖等信息
- `src/Main.ts` - 模组的主入口文件，实现模组的核心功能
- `README.md` - 模组的描述文档，包含模组的功能描述、使用方法、注意事项等，**应该同时包括中文和英文**
